Webhook Integration

Receive real-time notifications about system events.

Webhook Configuration

POST /api/webhooks/configure
{
  "url": "https://your-app.com/webhook",
  "events": ["document.processed", "user.created"],
  "secret": "your-webhook-secret"
}

Event Types

  • document.processed - Document processing completed
  • user.created - New user registered
  • query.processed - AI query completed
  • error.occurred - System error occurred

Authentication

Secure authentication methods for API access.

JWT Token Authentication

# Login to get token
curl -X POST https://your-domain.com/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username": "your-username", "password": "your-password"}'

# Use token in subsequent requests
curl -X GET https://your-domain.com/api/documents \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"

API Key Authentication

curl -X GET https://your-domain.com/api/documents \
  -H "X-API-Key: YOUR_API_KEY"

Code Examples

Practical examples for common integration scenarios.

JavaScript Integration

// Initialize RaghimAI client
const raghimAI = new RaghimAIClient({
  apiUrl: 'https://your-domain.com/api',
  token: 'your-jwt-token'
});

// Upload document
const uploadResult = await raghimAI.documents.upload(file);

// Send query
const response = await raghimAI.chat.send({
  message: 'What is this document about?',
  collectionId: uploadResult.collectionId
});

Python Integration

import requests

# Initialize client
api_url = 'https://your-domain.com/api'
headers = {'Authorization': 'Bearer your-jwt-token'}

# Upload document
with open('document.pdf', 'rb') as f:
    files = {'file': f}
    response = requests.post(f'{api_url}/documents/upload', 
                           files=files, headers=headers)

# Send query
query_data = {
    'message': 'Summarize this document',
    'collection_id': response.json()['collectionId']
}
response = requests.post(f'{api_url}/chat', 
                        json=query_data, headers=headers)
