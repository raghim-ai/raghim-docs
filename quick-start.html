Prerequisites

Ensure your system meets the minimum requirements before installation.

System Requirements

  • OS: Linux (Ubuntu 18.04+, CentOS 7+)
  • RAM: 8GB minimum
  • Storage: 20GB+ free space
  • CPU: 4 cores minimum

Software Requirements

  • Docker: 20.10+
  • Docker Compose: 2.0+
  • Python: 3.7+
  • Git: Latest version

Installation

Deploy RaghimAI Enterprise using the offline package.

1

Download Package

Download the offline enterprise package from your RaghimAI representative.

# Extract the package
tar -xzf rag-enterprise-1.0.26-offline.tar.gz
cd rag-enterprise-1.0.26-offline
2

Run Installer

Execute the interactive installer for guided setup.

# Launch interactive installer
./INSTALL.sh
Note: The installer will guide you through license acceptance and configuration.
3

Verify Installation

Check that all services are running correctly.

# Run health check
./scripts/health_check.sh

Configuration

Configure your RaghimAI Enterprise system for your environment.

Environment Variables

# Essential configuration
LICENSE_KEY=your-license-key
JWT_SECRET_KEY=your-secret-key
DATABASE_PASSWORD=secure-password
CACHE_PASSWORD=secure-password

# Optional configuration
GPU_ENABLED=true
FRONTEND_URL=http://localhost
API_URL=http://localhost:8000

Admin User Setup

# Create admin user
curl -X POST http://localhost:8000/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{
    "username": "admin",
    "password": "secure-password",
    "email": "admin@yourcompany.com",
    "role": "admin"
  }'

First Steps

Start using your RaghimAI Enterprise system.

1

Access the System

Open your browser and navigate to the frontend URL.

Frontend: http://localhost
API: http://localhost:8000
2

Login

Use your admin credentials to log into the system.

3

Upload Documents

Upload your first documents to start building your knowledge base.

  • Supported formats: PDF, DOCX, TXT, MD
  • OCR support for scanned documents
  • Automatic content extraction
4

Test AI Queries

Send your first AI query to test the system.

# Example query
"What is the main topic of the uploaded documents?"
