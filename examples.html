JavaScript Examples

Client-side integration examples using JavaScript.

Basic API Client

class RaghimAIClient {
  constructor(apiUrl, token) {
    this.apiUrl = apiUrl;
    this.token = token;
  }

  async request(endpoint, options = {}) {
    const response = await fetch(`${this.apiUrl}${endpoint}`, {
      headers: {
        'Authorization': `Bearer ${this.token}`,
        'Content-Type': 'application/json',
        ...options.headers
      },
      ...options
    });
    return response.json();
  }

  async uploadDocument(file) {
    const formData = new FormData();
    formData.append('file', file);
    
    const response = await fetch(`${this.apiUrl}/documents/upload`, {
      method: 'POST',
      headers: {
        'Authorization': `Bearer ${this.token}`
      },
      body: formData
    });
    return response.json();
  }

  async sendQuery(message, collectionId) {
    return this.request('/chat', {
      method: 'POST',
      body: JSON.stringify({
        message,
        collection_id: collectionId
      })
    });
  }
}

Usage Example

// Initialize client
const client = new RaghimAIClient('https://<your-domain>/api', 'your-jwt-token');

// Upload document
const fileInput = document.getElementById('file-input');
const file = fileInput.files[0];
const uploadResult = await client.uploadDocument(file);

// Send query
const response = await client.sendQuery(
  'What is this document about?',
  uploadResult.collection_id
);

console.log(response.message);

Python Examples

Server-side integration examples using Python.

Python Client

import requests
import json

class RaghimAIClient:
    def __init__(self, api_url, token):
        self.api_url = api_url
        self.token = token
        self.headers = {
            'Authorization': f'Bearer {token}',
            'Content-Type': 'application/json'
        }

    def upload_document(self, file_path):
        with open(file_path, 'rb') as f:
            files = {'file': f}
            headers = {'Authorization': f'Bearer {self.token}'}
            response = requests.post(
                f'{self.api_url}/documents/upload',
                files=files,
                headers=headers
            )
        return response.json()

    def send_query(self, message, collection_id):
        data = {
            'message': message,
            'collection_id': collection_id
        }
        response = requests.post(
            f'{self.api_url}/chat',
            json=data,
            headers=self.headers
        )
        return response.json()

    def get_documents(self):
        response = requests.get(
            f'{self.api_url}/documents',
            headers=self.headers
        )
        return response.json()

Usage Example

# Initialize client
client = RaghimAIClient('https://<your-domain>/api', 'your-jwt-token')

# Upload document
result = client.upload_document('document.pdf')
print(f"Document uploaded: {result['documentId']}")

# Send query
response = client.send_query(
    'Summarize this document',
    result['collection_id']
)
print(f"AI Response: {response['message']}")

# List documents
documents = client.get_documents()
for doc in documents:
    print(f"Document: {doc['name']}")

cURL Examples

Command-line integration examples using cURL.

Authentication

# Login to get token
curl -X POST https://<your-domain>/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{
    "username": "your-username",
    "password": "your-password"
  }'

Document Upload

# Upload document
curl -X POST https://<your-domain>/api/documents/upload \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -F "file=@document.pdf"

Send Query

# Send AI query
curl -X POST https://<your-domain>/api/chat \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "message": "What is this document about?",
    "collection_id": "your-collection-id"
  }'

Agent Flow Examples

Illustrative examples for triggering and monitoring agent flows via the REST API.

Trigger a Flow (cURL)

# List available flows
curl -X GET https://<your-domain>/api/agent/<flows-list-operation> \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"

# Trigger a flow run (streaming SSE response)
curl -X POST https://<your-domain>/api/agent/<flow-run-operation> \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "note": "Trigger payload schema available in approved technical onboarding"
  }'

Python — Trigger Flow

import requests

api_url = 'https://<your-domain>/api'
headers = {'Authorization': 'Bearer your-jwt-token'}

# Get available flows
flows = requests.get(f'{api_url}/agent/<flows-list-operation>', headers=headers)
flow_list = flows.json()

# Trigger a flow (note: real endpoint path from your deployment)
run_response = requests.post(
    f'{api_url}/agent/<flow-run-operation>',
    json={'note': 'payload per approved integration docs'},
    headers=headers,
    stream=True  # flows return streaming SSE
)
for line in run_response.iter_lines():
    if line:
        print(line.decode('utf-8'))

MCP Tool Examples

Illustrative examples for working with Model Context Protocol tools configured in your deployment.

List Available MCP Tools (cURL)

# List configured MCP servers and tools
curl -X GET https://<your-domain>/api/mcp-tools/<list-operation> \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"

MCP tools are invoked within agent flows — they are not called directly via the public REST API. Configure MCP servers through Admin → MCP Tools and reference them in flow node configurations.
